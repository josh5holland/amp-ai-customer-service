## Rule Categories

### Personal Safety Equipment

| Activity | Helmet | Suit | Gloves | Shoes |
|----------|--------|------|--------|-------|
| Road course track day | SA rated | Optional | Optional | Closed-toe |
| Road course racing | SA rated | Required | Required | Racing boots |
| Race karting | SA/M/K rated | Optional | Optional | Closed-toe |
| Concession karting | Provided | N/A | N/A | Closed-toe |

### Helmet Ratings Explained
- **SA (Special Application):** Motorsports, fire-resistant
- **M (Motorcycle):** Street motorcycle use
- **K (Karting):** Karting-specific

**Road course requires SA.** Karting accepts SA, M, or K.

## Flag Signals

| Flag | Action Required |
|------|-----------------|
| Green | Continue normally |
| Yellow | No passing, reduce speed |
| Red | Stop safely |
| Black | Report to pit immediately |
| Blue | Let faster car pass |
| White | Slow vehicle ahead |
| Checkered | Session complete |

## Vehicle Requirements Decision Tree

```
What type of event?
├── Track Day / HPDE → Basic tech (brakes, fluids, battery)
├── Time Attack → Basic tech + fire extinguisher
└── Club Racing → Full safety build (cage, harness, suppression)
```

## Common Rule Questions

| Question | Answer |
|----------|--------|
| "What helmet for track days?" | Snell SA2015 or SA2020 |
| "What helmet for karting?" | Snell SA, M, or K |
| "Do I need a roll cage?" | Only for wheel-to-wheel racing |
| "Fire extinguisher required?" | Required for competition, recommended for all |
| "Noise limits?" | Varies by event - contact AMP |
