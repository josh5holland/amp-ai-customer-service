# AMP Onboarding Coordinator - Claude Code Reference

## Purpose
Quick reference for AI assistants helping new members get started at AMP.

---

## Onboarding Paths

### Road Course Member

```
Join → Welcome Email → Portal Access → Calendar Review → First Visit
                                                              ↓
                                               Orientation → Tech → Track Time
```

### Race Karting Member

```
Join → Welcome Email → Portal Access → Karting School → Equipment
                                              ↓
                               Practice Days → First Race (3-6 months)
```

---

## First Visit Checklists

### Road Course
| Item | Required |
|------|----------|
| Helmet (Snell SA) | Yes |
| Closed-toe shoes | Yes |
| Long pants | Recommended |
| Tech-ready vehicle | Yes |
| Signed waiver | Yes |

### Race Karting
| Item | Required |
|------|----------|
| Helmet (SA/M/K) | Yes |
| Rib protector | Yes |
| Closed-toe shoes | Yes |
| Kart (or school arrangement) | Yes |
| Signed waiver | Yes |

## Key New Member Questions

| Question | Short Answer |
|----------|--------------|
| "What do I do now?" | Portal access → Calendar → Plan first visit |
| "When can I drive?" | Check member calendar |
| "What do I bring?" | Helmet, shoes, see checklist |
| "Do I register?" | Yes for some events, check calendar |
| "I'm nervous" | Normal! Start slow, ask for help |

## Onboarding Milestones

### Road Course
1. Membership confirmed
2. Portal access working
3. Calendar reviewed
4. First track day attended
5. Tech inspection passed
6. First session completed

### Race Karting
1. Membership confirmed
2. Karting School completed
3. Equipment acquired
4. Practice days (5-10 recommended)
5. First club race entered
6. First race completed

---

## Common Concerns

| Concern | Response Approach |
|---------|-------------------|
| "Never done track day" | Normalize, offer classroom, suggest starting slow |
| "Is my car ready?" | Provide tech checklist, suggest pre-check |
| "Don't know anyone" | Mention community focus, Track Day Socials |
| "Worried about safety" | Explain rules, flags, staff support |
| "Don't know procedures" | Walk through day structure, arrive early |
